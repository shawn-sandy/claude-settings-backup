# Sub-Agent Optimization

## Context

You are tasked with optimizing sub-agent implementations based on Claude Code documentation and best practices from: <https://docs.anthropic.com/en/docs/claude-code/sub-agents>

## Target

$ARGUMENTS

## Your Task

### Analysis Phase

1. **Examine Current Implementation**
   - Review the existing sub-agent code or configuration
   - Identify current functionality and limitations
   - Document performance bottlenecks or inefficiencies

2. **Best Practices Assessment**
   - Compare against Claude Code sub-agent guidelines
   - Check for proper tool usage patterns
   - Verify error handling and edge cases

### Optimization Phase

3. **Apply Optimizations**
   - Implement recommended patterns from documentation
   - Optimize tool usage and performance
   - Enhance error handling and robustness
   - Improve code clarity and maintainability

4. **Validation**
   - Test optimized implementation
   - Verify functionality is preserved
   - Document improvements made

## Optimization Criteria

- **Performance**: Reduce unnecessary operations and improve response times
- **Reliability**: Better error handling and edge case management
- **Maintainability**: Clear, well-structured code following conventions
- **Best Practices**: Adherence to Claude Code sub-agent guidelines
- **Tool Usage**: Efficient and appropriate tool selection

## Expected Deliverables

- Optimized sub-agent implementation
- Summary of improvements made
- Performance comparison (before/after)
- Updated documentation if applicable
